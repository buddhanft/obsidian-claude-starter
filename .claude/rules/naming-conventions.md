# 檔案命名規則

## 分類前綴（5 大類）

| 分類 | 涵蓋內容 |
|------|----------|
| **投資** | crypto, 地緣政治, 經濟, 主權, 市場分析 |
| **效率** | productivity, AI 工具, 工作方法, 時間管理 |
| **思維** | 哲學, 心理, 決策框架, 學習方法 |
| **技術** | 程式, 工具教學, 開發相關 |
| **健康** | biohack, 營養, 睡眠, 運動, 身體優化 |

## 命名規則

| Type | Pattern | Example |
|------|---------|---------|
| Inbox（一般） | `{分類}-{標題關鍵字}-{作者}-{YYYY-MM}.md` | `投資-主權是不良資產-mary-2026-01.md` |
| Inbox（書籍） | `書-{分類}-{書名關鍵字}-{作者}-{YYYY-MM}.md` | `書-思維-隨機騙局-taleb-2026-01.md` |
| Digest | `digest-{分類}-{標題關鍵字}-{作者}.md` | `digest-投資-主權是不良資產-mary.md` |
| Health Log | `YYYY-MM-DD-health-log.md` | `2026-01-10-health-log.md` |
| Goal | `goal-YYYY-{topic}.md` | `goal-2026-weight-management.md` |
| Weekly Review | `weekly-YYYY-WXX.md` | `weekly-2026-W02.md` |
| Monthly Health | `monthly-health-YYYY-MM.md` | `monthly-health-2026-01.md` |
| MOC | `moc-{topic}.md` | `moc-finance.md` |
| Position Thesis | `thesis-{TICKER}.md` | `thesis-BTC.md` |

## 命名原則

- **分類優先**：讓瀏覽時一眼看到主題
- **中文標題**：用簡短的中文描述核心內容
- **作者小寫**：統一用小寫英文
- **日期在後**：方便按主題排序，而非時間
- **日期使用擷取日期**：用「我處理這篇的月份」，不是原文發布日期
