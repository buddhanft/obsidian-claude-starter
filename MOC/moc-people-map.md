# People Map

> 不是 CRM。不是追蹤誰「有用」。
> 是一張誠實的地圖：誰讓我變更好，我讓誰變更好，哪些關係在安靜地死掉。

---

## Inner Circle（5 人以內）

讓我變成更好的人。不需要常見面，但每次互動都有份量。

| 誰 | 他給我什麼 | 我給他什麼 | 上次深度互動 | 溫度 |
|----|-----------|-----------|-------------|------|
| | | | | |

---

## Growth Circle（5~15 人）

現在還不夠深，但有潛力。或是曾經很近但正在冷卻的人。

| 誰 | 為什麼想靠近 | 障礙 | 一個具體下一步 |
|----|------------|------|--------------|
| | | | |

---

## Weak Ties（機會表面積）

不需要深交，但能帶來你不知道的東西。
Granovetter 的研究：新機會更常來自弱連結。

| 誰 | 他的世界跟我的交集在哪 | 上次接觸 |
|----|---------------------|---------|
| | | |

---

## 正在死掉的關係

曾經重要，但正在安靜地衰減。寫下來是為了做一個有意識的選擇：**要救還是要放？**

| 誰 | 曾經為什麼重要 | 為什麼在衰減 | 決定：救 or 放 |
|----|--------------|------------|--------------|
| | | | |

---

## 反勢利眼檢查

每次更新這份地圖時，問自己：

1. **我是在想「這個人對我有什麼用」嗎？** → 如果是，那段關係可能不值得放在這裡
2. **有沒有人我一直在提款但沒有存款？** → 想想你能給他什麼
3. **有沒有人你不「需要」但就是想見？** → 那可能是最重要的關係

---

## 與 Weekly Review 的整合

每週回顧時加一個問題：

> **這週有沒有一個讓我感到「被理解」或「被挑戰」的對話？跟誰？**

連續三週答案是「沒有」→ 看這張地圖，找一個人主動聯絡。
